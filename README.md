# Python Student Result Automation System

## Objective
To build a Python-based automation system for storing student marks, processing results, generating reports, and exporting result sheets.

## Tools Used
Python, SQLite, CSV, VS Code

## Features
- Menu-driven program
- Student data entry system
- Database integration using SQLite
- Average marks calculation
- Pass/Fail and grade system
- Top performer identification
- Automated CSV result export

## Outcome
Developed a complete result processing and automation system that demonstrates practical use of Python with databases and file handling.
